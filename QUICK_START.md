## 👥 第三步：审批审核员（管理员操作）

### 如果你是管理员

1. 使用 `admin/admin123` 登录
2. 点击左侧菜单"用户管理"
3. 在待审批列表中找到新用户
4. 点击"通过"按钮
5. 新用户现在可以登录了

---

## 📝 第四步：开始审核（审核员操作）

### 如果你是审核员

1. **登录系统**
   - 使用已审批的审核员账号登录

2. **领取任务**
   - 点击"领取新任务（20条）"按钮
   - 系统自动分配 20 条评论

3. **审核评论**
   - 仔细阅读评论内容
   - 选择"通过"或"不通过"
   - 如果不通过：
     - ✅ 必须选择至少一个违规标签
     - ✅ 必须填写不通过原因

4. **提交审核**
   - 单条提交：点击任务下方的"提交审核"
   - 批量提交：完成多条后点击"批量提交"

## 🔧 常见问题

### Q1: 登录后提示"登录已过期"
**A**: Token 默认有效期为 24 小时，过期后需要重新登录。

### Q2: 领取任务后没有显示任何内容
**A**: 
1. 检查数据库中是否有待审核的评论
2. 刷新页面重试
3. 查看浏览器控制台是否有错误

### Q3: 提交审核时提示"必须选择违规标签"
**A**: 当选择"不通过"时，必须：
- 至少选择一个违规标签
- 填写不通过原因

### Q4: 管理员在哪里审批用户？
**A**: 
1. 使用管理员账号登录
2. 点击左侧菜单"用户管理"
3. 在列表中点击"通过"按钮

### Q5: 如何添加新的违规标签？
**A**:
1. 管理员登录
2. 访问"标签管理"页面
3. 点击"新建标签"
4. 填写标签名称和描述

## 🔄 服务管理

### 启动服务

#### 方式一：一键启动（推荐）
```bash
start-all.bat
```

#### 方式二：分别启动

**后端：**
```bash
go run cmd/api/main.go
```

**前端：**
```bash
cd frontend
npm run dev
```

### 停止服务

在各自的命令行窗口按 `Ctrl + C`

### 重启服务

停止后重新运行启动命令
